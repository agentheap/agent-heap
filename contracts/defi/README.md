# DeFi Contracts

Ready-to-deploy DeFi smart contracts for freelance clients.

## Contracts

### SimpleStaking
Stake any ERC20 token to earn a reward token. Features:
- Per-second reward accrual
- Emergency withdrawal
- Owner-adjustable reward rate
- Use case: LP staking, farming pools, loyalty programs

### VestingWallet
Linear token vesting with cliff for team/investor allocations. Features:
- Multiple beneficiary schedules
- Cliff period support
- Revocable schedules (owner only)
- Use case: TGE lockups, team tokens, advisor allocations
