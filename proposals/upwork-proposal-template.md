# Upwork Proposal Template

Use this template when applying for Solidity/smart contract gigs on Upwork.

## Profile Bio

> Senior Solidity developer with X years of experience deploying ERC20, ERC721, and DeFi
> contracts on Ethereum, Polygon, BSC, Arbitrum, and Base. OpenZeppelin audited patterns,
> Hardhat/Foundry testing, Slither security analysis. Portfolio: github.com/yourhandle

## Proposal Structure

```
Subject: Quick turnaround, [N]+ years Solidity experience

Hi [Client Name],

I can definitely help with your [project type — ERC20 token / NFT collection / DeFi contract].
I've deployed similar contracts [N] times before.

Here's what I'd deliver:
1. Smart contract(s) — battle-tested, OpenZeppelin-based
2. Comprehensive test suite (Hardhat)
3. Deployment scripts + verification
4. Flat-file for verification + documentation

**My timeline:** [1-3] days for this scope
**Fixed price:** $[amount]
**Includes:** [one round of revisions / deployment support]

I've attached my portfolio with sample contracts. Happy to jump on a quick call
to discuss specifics.

Best,
[Your Name]
```

## Tips

- **Be specific** — reference their project description, not a generic pitch
- **Show, don't tell** — link to your GitHub portfolio
- **Price confidently** — don't undercut too much; $75-150 for an ERC20 with fees is reasonable
- **Include test coverage** — mention that your deliverables include tests
- **Ask one question** — "Do you need buy/sell taxes?" shows you know your stuff

## Follow-up

If they don't respond in 48 hours, send a gentle nudge:
> Hi [Client], just checking in — any questions about the proposal? Happy to
> clarify anything or adjust scope.
